Entwine: Forged to FlyProject 
Overview    
Developed a functional ornithopter wing-flapping model in SolidWorks.
Conducted aerodynamic analysis support and 3D design optimization.
Secured 3rd Position in the Entwine competition organized by NITK.
Technical DeliverablesCAD Models: Complete assembly and part files for the flapping mechanism.
Analysis Data: Support files for Ansys Fluent and CFD-Post evaluations.

Tech StackSoftwares: SolidWorks, AutoCAD, Ansys Fluent, CFD-Post.
Methodologies: 3D Modeling, Aerodynamic Shaping, Structural Analysis.
